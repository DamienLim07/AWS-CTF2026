# AWS-Skills-to-Jobs-CTF-2026
My participation in the AWS Skills to Jobs CTF 2026 powered by SANS, 
held from March 11–13, 2026 (ET). My display name for this event was "soupisdeli".

## Results
I ranked 2nd among Singaporean participants, confirmed directly by a 
SANS organiser. As my region was not registered in the system, this is 
not reflected on the official leaderboard. Supporting evidence including 
the regional leaderboard data and organiser confirmation have been 
included in this repo.

I scored 6,898 / 7,500 points, completing 35 / 37 challenges and placing 
59th globally. 6 hints were activated, 1 for the intro challenge and 5 
on an uncompleted challenge, adding no value to the final score.
